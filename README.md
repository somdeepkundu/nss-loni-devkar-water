# Crop Water Productivity Analysis - Loni Dewakar, Pune

Analyzing and improving crop water productivity in Loni Dewakar village, Pune district, using NSS (National Sample Survey) data and complementary agricultural datasets.

## Goal
Improve crop water productivity through data-driven insights on:
- Current water usage patterns
- Crop selection and yields
- Irrigation practices
- Water availability vs. demand

## Data Sources
- **NSS 77th Round (2019-20)**: Agricultural data for Pune district
- Climate/rainfall data for Pune
- Irrigation and water management records
- Local crop and soil data
